HOW TO TEST
-----------
# Build CLI
cargo build --release

# Start first peer (Alice)
./target/release/umbra start -u alice -p 9000 -t "test-room"

# Start second peer (Bob) - connect to Alice
./target/release/umbra start -u bob -p 9001 \
  -c "/ip4/127.0.0.1/udp/9000/quic-v1/p2p/<ALICE_PEER_ID>" \
  -t "test-room"

# Alice and Bob can now chat with E2E encryption!
